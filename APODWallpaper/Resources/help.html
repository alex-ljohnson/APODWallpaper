APOD Wallpaper Help & Documentation

Data location

All of the data files the app creates can be found under 'appdata%\APODWallpaper'.
This includes images, image metadata, api information caches, settings, etc.

Changing settings

The settings can be accessed either from the Configurator GUI (if installed) or by editing the config.json file in the data directory.

Additional Notes

  • The application requires an internet connection to download images and data from NASA's APOD API.
  • Additional information may be available on the Github README.md
  • For any issues or bugs, please refer to the GitHub repository or contact support (me@transparentbox.uk)

FAQs

Why aren't there any new images?

A: Because the NASA site may not be posting them (e.g. during US Gov. shutdown)

Do I need the configurator?

A: No, all the options are available in the config.json file, with documentation available. The configurator simply provides a user interface.
